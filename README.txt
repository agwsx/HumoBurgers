HUMO BURGERS

Descripción:
Sitio para promocionar e informar sobre los alimentos de Humo Burgers.

Objetivo:
Desarrollar un sitio web para curso de Coderhouse utilizando las herramientas aprendidas

Características:
Sitio web para navegar y brindar información sobre como contactarse.
El sitio contará con 5 páginas y un formulario para comentarios.

Construido con HTMLy SASS/CSS
Utilizando librerias de Bootstrap v5.1.3 para estilo y diseño mobile, y Animate.css v4.1.1 para agregar animaciones.
El sitio se mantiene en constante construcción y actualización

Para instalar y configurar:
Es necesario tener instalado Git version 2.37.2 
Luego clonar repositorio con:
git clone https://github.com/agwsx/HumoBurgers.git
